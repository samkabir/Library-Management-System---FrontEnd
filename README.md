### Installation Setup

1. Go to your preferred folder, and open terminal
```
git clone git@github.com:samkabir/Library-Management-System-FrontEnd.git
```

2. Go to the repository folder and open terminal, and run
```
npm install
```

3. Run the server
```
npm run dev
```
